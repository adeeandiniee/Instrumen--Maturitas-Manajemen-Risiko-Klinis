# Instrumen--Maturitas-Manajemen-Risiko-Klinis
Instrumen ini digunakan untuk menilai tingkat kematangan (maturitas) penerapan manajemen risiko klinis di RSAB Harapan Kita. Hasil pengukuran menjadi dasar evaluasi tahunan dan penyusunan program perbaikan berkelanjutan. Instrumen dapat diisi oleh seluruh staf, baik klinis maupun non klinis, sesuai peran masing-masing.
